Getting Started

The Admin Console Dashboard is a Google Admin's home base to access all users, devices, services and domain configurations. There are several important tasks that a new Google Admin should perform. We will review the top 3.


#1 - Set the primary admin email Admin Console > Account > Account Settings > Profile > Contact Info It is important to set this properly to ensure you receive important Google communication.

#2 - Create a proper Organization Unit Strcture Admin Console > Directory > Organizational Units Initially in your Google Admin console, all your users and devices are placed in a single organizational unit, called the top-level organizational unit. All settings you make in the Admin console apply to this top-level organizational unit and, therefore, to all users and devices in your account.

#3 - Decide how you're going to create your users Admin Console > Directory > Users You can manually create users, upload a file, or sync with another source.
